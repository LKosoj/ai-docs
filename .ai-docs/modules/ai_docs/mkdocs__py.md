# ai_docs/mkdocs

```markdown
Модуль предоставляет инструменты для генерации конфигурационного файла `mkdocs.yml` и записи Markdown-документации на диск. Поддерживает динамическое построение навигации по разделам, конфигурациям, модулям и другим компонентам проекта. Использует YAML-сериализацию с кастомными тегами для интеграции с Mermaid-диаграммами через mkdocs-plugin.

Ключевые структуры данных
_YamlPythonName — строковый тип для представления Python-имён в YAML с кастомным тегом
_YamlSafeDumper — кастомный YAML-дампер с поддержкой тега !python/name

build_mkdocs_yaml(site_name: str, sections: Dict[str, str], configs: Dict[str, str], local_site: bool = False, has_modules: bool = False, module_nav_paths: List[str] | None = None, project_config_nav_paths: List[str] | None = None) -> str
Генерирует содержимое файла mkdocs.yml в виде строки
Аргументы
site_name — название сайта документации
sections — словарь включённых разделов (ключ — имя, значение — путь к файлу)
configs — словарь конфигурационных разделов (например, kubernetes, helm и т.д.)
local_site — если True, отключает site_url и use_directory_urls для локального просмотра
has_modules — указывает, что проект содержит модули, требующие навигации
module_nav_paths — список путей к файлам модулей для построения древовидной навигации
project_config_nav_paths — список путей к файлам конфигурации проекта для навигации
Возвращает
Строка в формате YAML, соответствующая схеме mkdocs.yml
Исключения
Не выбрасывает исключения

---
_build_tree_nav(paths: List[str], strip_prefix: str) -> List[Dict[str, object]]
Строит древовидную структуру навигации из списка путей
Аргументы
paths — список относительных путей к документам
strip_prefix — префикс, который будет удалён из начала каждого пути
Возвращает
Список словарей, представляющих древовидную навигацию
Исключения
Не выбрасывает исключения

---
_insert_nav_node(tree: Dict[str, object], parts: List[str], rel_path: str) -> None
Рекурсивно вставляет узел в дерево навигации по частям пути
Аргументы
tree — изменяемое дерево навигации
parts — части пути, по которым строится вложенность
rel_path — исходный относительный путь к файлу
Возвращает
Ничего не возвращает
Исключения
Не выбрасывает исключения

---
_tree_to_nav(tree: Dict[str, object]) -> List[Dict[str, object]]
Преобразует внутреннее дерево навигации в список словарей для mkdocs.yml
Аргументы
tree — словарь с вложенной структурой навигации
Возвращает
Список словарей, отсортированный по алфавиту, с папками в начале
Исключения
Не выбрасывает исключения

---
write_docs_files(docs_dir: Path, files: Dict[str, str]) -> None
Записывает файлы документации на диск в указанную директорию
Аргументы
docs_dir — корневая директория для записи файлов
files — словарь, где ключ — относительный путь, значение — содержимое файла
Возвращает
Ничего не возвращает
Исключения
Может выбросить исключение при ошибках записи на диск (например, PermissionError)
```
