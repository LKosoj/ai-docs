# ai_docs/mkdocs

Модуль предоставляет инструменты для генерации конфигурационного файла `mkdocs.yml` и записи Markdown-документации в файловую систему. 
Поддерживается формирование навигации по разделам документации, включая модули, конфигурации и проектные настройки, с иерархической структурой. 
Используется кастомный YAML-дампер для корректного представления Python-специфичных тегов, таких как `!python/name`.

Ключевые структуры данных
_YamlPythonName — строковый тип для представления Python-имён в YAML через тег `!python/name`
_YamlSafeDumper — безопасный YAML-дампер с поддержкой кастомного представления Python-имён

build_mkdocs_yaml(site_name: str, sections: Dict[str, str], configs: Dict[str, str], local_site: bool = False, has_modules: bool = False, module_nav_paths: List[str] | None = None, project_config_nav_paths: List[str] | None = None) -> str
Генерирует содержимое файла mkdocs.yml в формате YAML
Аргументы
site_name — название сайта документации
sections — словарь включённых разделов (ключ — имя, значение — путь к файлу)
configs — словарь конфигурационных разделов (ключ — тип конфига, значение — имя файла)
local_site — если True, отключает публичные URL и включает flat-режим адресов
has_modules — указывает, что проект содержит модули, требующие навигации
module_nav_paths — список путей к файлам модулей для построения иерархии
project_config_nav_paths — список путей к файлам конфигурации проекта для навигации
Возвращает
Строка в формате YAML, представляющая конфигурацию MkDocs
Исключения
Не выбрасывает исключения

_build_tree_nav(paths: List[str], strip_prefix: str) -> List[Dict[str, object]]
Строит иерархическую структуру навигации из списка путей
Аргументы
paths — список относительных путей к документам
strip_prefix — префикс, который будет удалён из начала каждого пути
Возвращает
Список словарей, представляющих иерархию навигации
Исключения
Не выбрасывает исключения

_insert_nav_node(tree: Dict[str, object], parts: List[str], rel_path: str) -> None
Рекурсивно вставляет узел в дерево навигации по частям пути
Аргументы
tree — изменяемое дерево навигации
parts — части пути, по которым строится вложенность
rel_path — исходный относительный путь к файлу
Возвращает
Ничего не возвращает
Исключения
Не выбрасывает исключения

_tree_to_nav(tree: Dict[str, object]) -> List[Dict[str, object]]
Преобразует внутреннее дерево навигации в список словарей для MkDocs
Аргументы
tree — словарь с вложенной структурой навигации
Возвращает
Список словарей, пригодных для использования в `nav` MkDocs
Исключения
Не выбрасывает исключения

write_docs_files(docs_dir: Path, files: Dict[str, str]) -> None
Записывает файлы документации в указанную директорию
Аргументы
docs_dir — корневая директория для записи файлов документации
files — словарь, где ключ — относительный путь, значение — содержимое файла
Возвращает
Ничего не возвращает
Исключения
Может выбросить исключение при ошибках ввода-вывода (например, PermissionError)
