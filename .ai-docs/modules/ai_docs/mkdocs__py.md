# ai_docs/mkdocs

```markdown
Модуль предоставляет инструменты для генерации конфигурационного файла `mkdocs.yml` и записи документационных файлов на диск.  
Основной функционал включает построение навигации по документации на основе структуры проекта, поддержку модулей и конфигураций,  
а также корректную сериализацию специальных Python-сущностей в YAML. Также реализована вложенная навигация через древовидную структуру.

Ключевые структуры данных
_YamlPythonName — строковый тип для представления Python-имён в YAML с использованием тега `python/name`
_YamlSafeDumper — кастомный YAML-дампер, расширяющий SafeDumper для поддержки Python-имён

build_mkdocs_yaml(site_name: str, sections: Dict[str, str], configs: Dict[str, str], local_site: bool = False, has_modules: bool = False, module_nav_paths: List[str] | None = None, project_config_nav_paths: List[str] | None = None) -> str
Генерирует содержимое файла mkdocs.yml в виде строки
Аргументы
site_name — название сайта документации
sections — словарь секций документации, где ключ — имя секции, значение — путь к файлу
configs — словарь конфигурационных разделов (например, kubernetes, helm) и соответствующих им имён файлов
local_site — если True, отключает site_url и use_directory_urls для локального просмотра
has_modules — указывает, есть ли в проекте модули, требующие отдельной секции в навигации
module_nav_paths — список путей к файлам модулей для построения древовидной навигации
project_config_nav_paths — список путей к файлам конфигурации проекта для древовидной навигации
Возвращает
Строка в формате YAML, представляющая конфигурацию MkDocs
Исключения
Не выбрасывает исключения

---
_build_tree_nav(paths: List[str], strip_prefix: str) -> List[Dict[str, object]]
Строит древовидную структуру навигации на основе списка путей
Аргументы
paths — список относительных путей к файлам документации
strip_prefix — префикс, который будет удалён из начала каждого пути
Возвращает
Список словарей, представляющих древовидную навигацию
Исключения
Не выбрасывает исключения

---
_insert_nav_node(tree: Dict[str, object], parts: List[str], rel_path: str) -> None
Рекурсивно вставляет узел в дерево навигации по частям пути
Аргументы
tree — изменяемое дерево навигации
parts — части пути, по которым определяется вложенность
rel_path — исходный относительный путь к файлу
Возвращает
Ничего не возвращает
Исключения
Не выбрасывает исключения

---
_tree_to_nav(tree: Dict[str, object]) -> List[Dict[str, object]]
Преобразует внутреннее дерево навигации в список словарей для MkDocs
Аргументы
tree — словарь с вложенной структурой навигации
Возвращает
Список словарей, пригодных для использования в mkdocs.yml
Исключения
Не выбрасывает исключения

---
write_docs_files(docs_dir: Path, files: Dict[str, str]) -> None
Записывает документационные файлы на диск в указанную директорию
Аргументы
docs_dir — путь к директории, куда будут записаны файлы
files — словарь, где ключ — относительный путь файла, значение — его содержимое
Возвращает
Ничего не возвращает
Исключения
Может выбросить исключение при ошибках ввода-вывода (например, PermissionError)
```
