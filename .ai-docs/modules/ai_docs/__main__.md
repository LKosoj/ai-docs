# ai_docs/__main__

```markdown
Модуль обеспечивает запуск основного приложения из командной строки, корректно обрабатывая импорты как при обычном использовании пакета, так и при запуске в качестве скрипта.  
Он динамически подгружает функцию `main` из модуля `cli`, при необходимости модифицируя `sys.path` для разрешения зависимостей.  
Это позволяет запускать приложение из корня проекта или через `python -m ai_docs`.

Ключевые структуры данных  
main — Основная функция запуска CLI-приложения, загружается динамически при инициализации модуля

_load_main()
Загружает и возвращает функцию main из модуля ai_docs.cli
Аргументы
Возвращает
Функция main из модуля cli
Исключения
ImportError — Возникает, если модуль не найден при прямом импорте; обрабатывается через fallback-механизм

---
main()
Точка входа в приложение; вызывается при запуске модуля как скрипта
Аргументы
Возвращает
Исключения
```
