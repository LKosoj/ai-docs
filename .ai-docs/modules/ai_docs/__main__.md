# ai_docs/__main__

```markdown
Модуль обеспечивает загрузку и запуск основной функции CLI-приложения `ai_docs` с поддержкой запуска как из пакета, так и напрямую как скрипта.  
Он корректно обрабатывает случаи, когда модуль импортируется в стандартном режиме и когда выполняется через `python <путь_к_модулю>`.  
Использует динамическую настройку `sys.path` для обеспечения доступности пакета в обоих сценариях.

_load_main()
Загружает и возвращает функцию main из модуля ai_docs.cli, обеспечивая совместимость с различными способами запуска
Возвращает
Функция main из ai_docs.cli
Исключения
ImportError — если модуль не найден и требуется fallback-импорт

---
main
Точка входа в CLI-приложение ai_docs
```
