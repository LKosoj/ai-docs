# ai_docs/domain

Модуль предоставляет средства классификации файлов по типу содержимого и доменам инфраструктуры. Определяет категорию файла (код, конфигурация, документация и др.) и выявляет домены, связанные с инфраструктурой (Kubernetes, Terraform, CI/CD и т.д.). Используется для анализа структуры проекта и автоматической категоризации файлов.

Ключевые структуры данных  
Path — объект пути к файлу из модуля pathlib  
Set[str] — множество строк, используемое для возврата набора доменов

classify_type(path: Path) -> str  
Определяет тип файла на основе расширения или имени.  
Аргументы  
path — путь к файлу для классификации  
Возвращает  
Строку, описывающую тип файла: "code", "docs", "config", "data", "ci", "infra" или "other"

---
detect_domains(path: Path, content_snippet: str) -> Set[str]  
Определяет домены инфраструктуры, связанные с файлом, на основе имени, пути и фрагмента содержимого.  
Аргументы  
path — путь к файлу  
content_snippet — фрагмент содержимого файла для анализа  
Возвращает  
Множество строк, представляющих домены (например, "kubernetes", "docker", "ci")

---
is_infra(domains: Set[str]) -> bool  
Проверяет, относится ли файл к инфраструктурным доменам.  
Аргументы  
domains — множество доменов, полученных из detect_domains  
Возвращает  
True, если хотя бы один из доменов относится к инфраструктуре, иначе False
