# ai_docs/__main__

Модуль обеспечивает запуск основного приложения из командной строки, корректно обрабатывая импорты как при обычном использовании пакета, так и при запуске в качестве скрипта. Он динамически подгружает функцию `main` из модуля `cli`, при необходимости модифицируя `sys.path` для разрешения зависимостей. Это позволяет запускать приложение из корня проекта или как установленный пакет.

Ключевые структуры данных  
main — Основная функция запуска CLI-приложения, загружаемая динамически

_load_main()
Загружает и возвращает функцию main из модуля ai_docs.cli
