# ai_docs/changes

```markdown
Модуль предоставляет функцию формирования Markdown-отчёта о результатах генерации файлов.  
Отчёт включает списки добавленных, изменённых и удалённых файлов, перечень перегенерированных разделов и краткое резюме.  
Данные структурированы для удобного восприятия с использованием заголовков и маркированных списков.

Ключевые структуры данных  
added — словарь добавленных файлов, где ключи — пути к файлам  
modified — словарь изменённых файлов, где ключи — пути к файлам  
deleted — словарь удалённых файлов, где ключи — пути к файлам  
regenerated_sections — список названий разделов, которые были перегенерированы  
summary — краткое текстовое резюме изменений

str format_changes_md(added: Dict, modified: Dict, deleted: Dict, regenerated_sections: List[str], summary: str)
Формирует Markdown-отчёт о изменениях в файлах и разделах
Аргументы
added — словарь добавленных файлов (ключи — пути)
modified — словарь изменённых файлов (ключи — пути)
deleted — словарь удалённых файлов (ключи — пути)
regenerated_sections — список названий перегенерированных разделов
summary — краткое резюме изменений
Возвращает
Строка в формате Markdown, содержащая отчёт об изменениях
Исключения
Отсутствуют
```
