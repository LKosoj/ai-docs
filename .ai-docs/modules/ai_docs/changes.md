# ai_docs/changes

```markdown
Функция формирует Markdown-отчёт о изменениях в проекте, включая добавленные, изменённые и удалённые файлы, 
перегенерированные разделы и краткое резюме. Отчёт структурирован с использованием заголовков второго уровня 
и маркированных списков. Предназначена для автоматической генерации отчётов в CI/CD или после выполнения скриптов обновления.

format_changes_md(added: Dict, modified: Dict, deleted: Dict, regenerated_sections: List[str], summary: str) -> str
Формирует Markdown-отчёт о типах изменений в файлах и разделах
Аргументы
added — словарь добавленных файлов, где ключи — пути к файлам
modified — словарь изменённых файлов, где ключи — пути к файлам
deleted — словарь удалённых файлов, где ключи — пути к файлам
regenerated_sections — список названий разделов, которые были перегенерированы
summary — краткое текстовое резюме изменений
Возвращает
строку в формате Markdown, содержащую структурированный отчёт об изменениях
```
