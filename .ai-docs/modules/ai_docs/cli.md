# ai_docs/cli

```markdown
Модуль `main` — это точка входа в инструмент автоматической генерации документации для репозиториев кода или конфигураций. Он объединяет функциональность сканирования исходных файлов, вызова LLM для генерации описаний и создания документации в формате README и/или сайта MkDocs. Поддерживает настройку через аргументы командной строки и переменные окружения, работу с локальными и удалёнными репозиториями, кэширование результатов и параллельную обработку.

parse_args() -> argparse.Namespace
Парсит аргументы командной строки для настройки генерации документации.
Возвращает
Объект с аргументами командной строки (Namespace)

---
resolve_output(source: str, output: Optional[str], repo_name: str) -> Path
Определяет путь к выходной директории для генерации документации.
Аргументы
source — Путь к исходной директории или URL репозитория
output — Указанный пользователем путь к выходной директории, может отсутствовать
repo_name — Имя репозитория, используемое при генерации пути по умолчанию
Возвращает
Полный путь к выходной директории (Path)

---
main() -> None
Основная функция, запускающая процесс генерации документации.
Выполняет загрузку переменных окружения, парсинг аргументов, сканирование исходников, инициализацию LLM и вызов генератора документации.
```
