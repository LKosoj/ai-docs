# tests/test_changes

Модуль предоставляет функциональность для форматирования информации об изменениях в документации в виде Markdown-текста. Основная функция `format_changes_md` генерирует структурированное описание добавленных, изменённых, удалённых файлов и перегенерированных разделов, а также включает краткое резюме изменений. Модуль используется для автоматической фиксации изменений в документации при её обновлении.

---

format_changes_md(added: dict, modified: dict, deleted: dict, regenerated: list, summary: str) -> str
Форматирует информацию об изменениях в виде Markdown-строки.
Аргументы
added — словарь добавленных файлов, где ключи — имена файлов
modified — словарь изменённых файлов, где ключи — имена файлов
deleted — словарь удалённых файлов, где ключи — имена файлов
regenerated — список названий перегенерированных разделов документации
summary — краткое текстовое резюме изменений
Возвращает
Markdown-строка, содержащая отформатированное описание всех изменений
