# Зависимости

`ai_docs` требует следующие внешние зависимости для корректной работы. Установка осуществляется через `pip` как часть пакета.

## Основные зависимости

| Пакет | Назначение | Минимальная версия |
|------|-----------|-------------------|
| `tiktoken` | Подсчёт и разбиение текста на токены для LLM | `0.7.0` |
| `PyYAML` | Генерация и парсинг `mkdocs.yml` | `6.0` |
| `requests` | HTTP-запросы к LLM API | `2.28.0` |
| `colorama` | Кросс-платформенная раскраска логов в терминале | `0.4.6` |

## Опциональные зависимости

- `mkdocs` — требуется при использовании флага `--mkdocs` для сборки статического сайта документации.
- `mkdocs-mermaid2-plugin` — поддержка диаграмм Mermaid в сгенерированной документации.

## Установка

```bash
pip install ai_docs
```

Для генерации сайта документации дополнительно установите:

```bash
pip install mkdocs mkdocs-mermaid2-plugin
```

## Управление окружением

Рекомендуется использовать виртуальное окружение:

```bash
python -m venv .venv
source .venv/bin/activate  # Linux/macOS
# или
.venv\Scripts\activate     # Windows

pip install ai_docs mkdocs mkdocs-mermaid2-plugin
```

## Проверка установки

После установки убедитесь, что CLI доступен:

```bash
ai_docs --help
```

Если команда не найдена, проверьте, что `pip` установил пакет в корректный `PATH`.
