ai_docs/utils

Модуль предоставляет набор утилит для работы с файлами, путями, хешированием и проверкой типов данных.
Основные функции включают вычисление SHA-256 хешей, безопасное создание слагов, определение двоичных файлов и проверку URL.
Поддерживает работу с объектами Path из pathlib, обеспечивая кроссплатформенную совместимость.

Ключевые структуры данных
Path — объект для удобной и безопасной работы с файловыми путями в кроссплатформенной среде

sha256_bytes(data: bytes) -> str
Вычисляет SHA-256 хеш от байтовых данных и возвращает его в виде шестнадцатеричной строки
Аргументы
data — входные данные в виде байтов
Возвращает
Шестнадцатеричное строковое представление хеша SHA-256

sha256_text(text: str) -> str
Вычисляет SHA-256 хеш от текстовой строки, кодированной в UTF-8
Аргументы
text — входная текстовая строка
Возвращает
Шестнадцатеричное строковое представление хеша SHA-256

read_text_file(path: Path) -> str
Считывает содержимое текстового файла с указанным путём, используя кодировку UTF-8
Аргументы
path — путь к файлу в виде объекта Path
Возвращает
Содержимое файла в виде строки
Исключения
OSError — если файл не существует или недоступен для чтения

safe_slug(path: str) -> str
Преобразует строку в безопасный слаг, заменяя неалфавитно-цифровые символы на подчёркивания
Аргументы
path — входная строка для преобразования
Возвращает
Нормализованная строка, пригодная для использования в именах файлов или URL

ensure_dir(path: Path) -> None
Создаёт директорию по указанному пути, включая родительские, если они отсутствуют
Аргументы
path — путь к директории, которую необходимо создать
Исключения
OSError — если директорию не удаётся создать по системным причинам

is_binary_file(path: Path, sample_size: int = 2048) -> bool
Определяет, является ли файл двоичным, на основе наличия нулевых байтов в первых байтах
Аргументы
path — путь к файлу
sample_size — количество байтов для анализа (по умолчанию 2048)
Возвращает
True, если файл, вероятно, двоичный; иначе False
Исключения
OSError — возвращает True при ошибках доступа к файлу

is_url(value: str) -> bool
Проверяет, является ли строка URL (HTTP/HTTPS/Git)
Аргументы
value — строка для проверки
Возвращает
True, если строка начинается с http://, https:// или git@; иначе False

to_posix(path: Path) -> str
Преобразует путь в строку в формате POSIX, независимо от ОС
Аргументы
path — объект Path для преобразования
Возвращает
Строка пути с косыми чертами (/) в стиле POSIX
