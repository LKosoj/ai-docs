Зависимости

Основные зависимости (pyproject.toml)

Зависимости проекта определены в pyproject.toml и включают следующие ключевые пакеты:

dependencies = [
    "requests",
    "tiktoken",
    "PyYAML",
    "mkdocs",
    "mkdocs-material",
    "mkdocs-mermaid2-plugin",
    "python-dotenv"
]
  • requests — для HTTP-запросов к LLM API.
  • tiktoken — подсчёт токенов и разбиение текста на чанки с учётом ограничений модели.
  • PyYAML — парсинг конфигурационного файла .ai-docs.yaml.
  • mkdocs, mkdocs-material, mkdocs-mermaid2-plugin — генерация и рендеринг сайта документации с поддержкой диаграмм Mermaid.
  • python-dotenv — загрузка переменных окружения из .env.

Системные зависимости

  • git — обязательно установлен в системе. Используется для клонирования удалённых репозиториев по URL.
  • Python ≥ 3.8 — минимальная поддерживаемая версия.

Опциональные зависимости

  • setuptools, wheel — требуются только при сборке пакета (указаны в build-system.requires).

Внешние инструменты

  • OpenAI-совместимый API — для работы LLM-клиента. Поддерживаются как официальные эндпоинты OpenAI, так и сторонние (через OPENAI_BASE_URL).
  • dotenv-файл — рекомендуется для хранения OPENAI_API_KEY и других параметров.

Управление зависимостями

Для установки зависимостей в режиме разработки:

pip install -e .

Зависимости автоматически устанавливаются при установке пакета через pip. Сборка пакета требует наличия setuptools>=68 и wheel.
