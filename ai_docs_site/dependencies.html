Зависимости

Основные зависимости (pyproject.toml)

Зависимости определены в секции [project.dependencies] и необходимы для работы всех компонентов ai_docs:

  • requests — HTTP-запросы к LLM API.
  • tiktoken — токенизация текста для контроля длины контекста.
  • PyYAML — парсинг конфигурационных файлов .ai-docs.yaml.
  • python-dotenv — загрузка переменных окружения из .env.
  • pathspec — обработка шаблонов игнорирования (.gitignore, .build_ignore).
  • mkdocs — генерация статического сайта документации.
  • mkdocs-mermaid2-plugin — поддержка диаграмм Mermaid в MkDocs.
  • mkdocs-pymdown-extensions — расширенные Markdown-функции (подсветка, таблицы и др.).

Опциональные зависимости

  • git (системный) — клонирование удалённых репозиториев по URL. Должен быть установлен в системе.

Управление зависимостями

Установка

pip install -e .

Команда устанавливает пакет в режиме разработки с зависимостями из pyproject.toml.

Проверка

Убедитесь, что установлены: - Python ≥ 3.8 - git в PATH (для работы с URL-репозиториями)

Контроль версий

Версии зависимостей не зафиксированы жёстко — используются минимальные совместимые версии. Для воспроизводимости используйте pip freeze > requirements.txt после установки.

Особенности использования

  • Кэширование LLM-запросов (tiktoken, pathspec) критично для производительности.
  • mkdocs и его плагины требуются только при генерации сайта (--mkdocs).
  • python-dotenv автоматически загружает .env при запуске main.py.

Системные требования

  • Операционная система: Linux, macOS, Windows (с поддержкой Python и git).
  • Свободное место: минимум 100 МБ (с учётом временных файлов и кэша).
